# DATA607 Assignments

This repository contains three assignments completed as part of the DATA 607 course, focused on applied machine learning and statistical modeling in Python.

## Overview

The assignments progress from foundational data exploration and preprocessing to more advanced modeling techniques and evaluation. Each notebook is self-contained and includes code, outputs, and brief interpretations.

## Files

**A1 – Data Exploration & Preprocessing**  
Initial analysis of the dataset, including cleaning, feature understanding, and exploratory data analysis (EDA).

**A2 – Modeling & Evaluation**  
Implementation of supervised learning models, with a focus on training, validation, and performance comparison.

**A3 – Advanced Modeling**  
Extension into more advanced techniques, including improved model selection, tuning, and deeper evaluation of results.

## Tools & Libraries

- Python (Jupyter Notebook)
- pandas, numpy
- scikit-learn
- matplotlib / seaborn

## Purpose

These assignments demonstrate:

- End-to-end data science workflow  
- Model building and evaluation  
- Ability to interpret and communicate results
